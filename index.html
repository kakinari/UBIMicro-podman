UBIMicro-podman

Red Hat UBI Micro をベースにした各種コンテナイメージをまとめたリポジトリです。 日本語環境、Java、Node.js、データベース、開発ツールなど、用途別に利用できるイメージを公開しています。

GitHub Repository Docker Hub

このリポジトリについて

ネットワーク上の VM や WSL で使いやすい開発・実行環境を、コンテナイメージとしてすばやく利用できるようにすることを目的としています。 学習用途、ビルド用途、CI/CD の実行環境としても活用できます。

軽量なベース環境

Red Hat UBI Micro ベースで、用途に応じて拡張しやすい構成です。

日本語利用を意識

日本語環境やフォント構成を考慮したイメージを提供しています。

用途別イメージ

Java、Node.js、DB、ビルド環境などを用途別ディレクトリで整理しています。

利用方法

Docker / Podman のどちらからでも Docker Hub 上のイメー���を取得できます。

Podman

podman pull docker.io/kakinari/ubi-micro-ja:tagname

Docker

docker pull kakinari/ubi-micro-ja:tagname

主なイメージ

リポジトリ内の代表的なイメージを抜粋しています。詳細は各ディレクトリを参照してください。

ubi-micro-9

RHEL 9 UBI Micro ベースの日本語対応ベースイメージ。

ubi-micro-base

C / C++ ベース開発向け。X-Window アプリ実行も想定した構成。

ubi-micro-jbase

Java アプリケーション向けベースイメージ。

ubi-micro-node

Node.js 利用向けのイメージ。

ubi-micro-postgres

PostgreSQL 関連の実行・検証用途向けイメージ。

ubi-micro-mysql

MySQL 関連の利用を想定したイメージ。

ubi-micro-dev

Development Tools を含むビルド向けイメージ。

ubi-micro-cs-nodejs

Codespace / Node.js 系の利用を想定した構成。

ディレクトリ構成の見方

各ディレクトリには対応するコンテナイメージの作成スクリプトや設定ファイルが含まれています。 利用したいイメージのフォルダに移動し、README や Dockerfile、ビルドスクリプトを参照してください。
